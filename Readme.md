This project is a prototype that the is built to help people quickly be safe during disasters like EARTHQUAKES,FLOODS,CYCLONES AND FIRE BREAKS.

The user can choose among 4 options mainly earthquake,cyclone,floods and fire breaks.

As the syste needs more contextual understanding, user also needs to choose whether he/she is indoor/outdoor.

Based on the input a detailed step by step guidance will be shown via intuitive images to guide the user facing the disaster.


A GPS feature is added that fetches nearby safe locations mainly police stations,hosplitals and fire stations.


A panic button is added so that when the situation is critical it will automatically activate itself and calls one of emergency numbers listed.
